# Palo Alto Firewall Rule Cleanup Automation

This project demonstrates a simple Python-based automation for generating a firewall
rule cleanup report. The script outputs a sample CSV containing rule names,
applications, actions, and hit counts.

### Features
- Demo script for rule analysis
- Exports sample rule data into CSV format
- Shows how automation supports firewall hygiene

### Tech Used
- Python
- CSV module

### How to run
```bash
python pa_rule_cleanup.py
